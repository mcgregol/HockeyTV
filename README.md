# htv-grabber
Download full game video from HockeyTV for free

## Dependencies
- Firefox
- ### On linux
    - install tk with your package manager
    - install geckodriver


- yt-dlp
    - ```pip install yt-dlp```
- selenium-wire
    - ```pip install selenium-wire```
- appJar
    - ```pip install appjar```

Easily install with ```pip install -r src/requirements.txt```

```ffmpeg``` recommended but not required

## Usage
```python src/main.py```
