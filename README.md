# htv-grabber
Download full game video from HockeyTV for free

## Dependencies
- Firefox
- On linux:
    - install ```tk``` with your package manager
- yt-dlp
    - ```pip install yt-dlp```
- selenium-wire
    - ```pip install selenium-wire```

Easily install with ```pip install -r src/requirements.txt```

## Usage
```python src/main.py```
